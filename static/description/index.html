Hide Product Cost

Hide cost price field from all users. Cost price remains visible to admin users and users with Purchase or Accounting access.

Overview

This module hides the Cost Price (standard_price) field on product forms from all users. The cost price remains visible to admin users and users with Purchase or Accounting access.

This is useful when you want to prevent all users from seeing product costs while still allowing purchasing and accounting teams full visibility.

Key Features

Automatic Hiding

No manual configuration needed. Install and the cost field is automatically hidden from unauthorized users.

Group-Based Access

Uses Odoo's built-in security groups to control visibility. No custom groups required.

Covers Both Views

Hides cost on both product.template and product.product form views.

Who Can See the Cost Price?

After installation, the cost price field will only be visible to users who belong to at least one of these groups:

  • Purchase / User (purchase.group_purchase_user)
  • Accountant (account.group_account_user)

Users without Purchase or Accounting access will not see the cost price field on product forms.

Dependencies

This module requires:

  • product - Product management (included in Odoo core)
  • purchase - Purchase module
  • account - Invoicing module

Installation

  1. Copy the product_hide_cost folder to your Odoo addons directory
  2. Update the apps list: Apps menu, then click Update Apps List
  3. Search for "Hide Product Cost" and click Install

No additional configuration required. The cost price field is immediately hidden from users without Purchase or Accounting access.
